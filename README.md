Objetivo del Repositorio
----------------------------------------------------------------
Este repositorio reúne todas las evidencias correspondientes al Primer Parcial (tareas 1 a la 4). Su objetivo principal es registrar y plantear una propuesta de solución tecnológica y de procesos, utilizando BPMN, ante la necesidad de un cambio en sus productos para la mejora nutricional de Barcel.

Estructura del Proyecto
----------------------------------------------------------------
El repositorio está organizado de la siguiente manera:

* Documentos : Contiene los documentos de investigación preliminar, el diagnóstico del problema (T1), el catálogo de reglas de negocio (T3) y el reporte de indagación sobre sistemas ERP/CRM/B2B/B2C (T4).
* Diagrama: Incluye el modelado del proceso "Planeación Financiera Estratégica y Gestión de Crisis" en formato de imagen ( -png) utilizando la notación BPMN 2.0 (T2).
* Entregable final : Contiene el documento integrador final en PDF correspondiente a la Tarea 5, el cual unifica todas las fases del proyecto.
